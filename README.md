# Desafío  Condiciones 

Este repositorio contiene la resolución del **Desafío 2**, enfocado en validar conocimientos de manipulación del DOM mediante JavaScript, control de flujos con estructuras condicionales (`if`, `else if`, `else`), operaciones aritméticas y operadores lógicos.

## 🛠️ Contenido del Desafío

### 1. Parte 1: Borde de Imagen
* **Requerimiento:** Al hacer click sobre una imagen única, se le añade un borde rojo de 2px. Al hacer click nuevamente, el borde se remueve.
* **Archivos:** `index1.html`, `script1.js`

### 2. Parte 2: Contador de Stickers
* **Requerimiento:** Permite ingresar cantidades para 3 tipos de stickers. Al verificar, si la suma total es menor o igual a 10 muestra el resultado (`Llevas X stickers`). Si excede el límite de 10, muestra una alerta dinámica (`Llevas demasiados stickers`).
* **Archivos:** `index2.html`, `script2.js`

### 3. Parte 3: Verificación de Password
* **Requerimiento:** Una interfaz con 3 menús desplegables (`<select>`) con dígitos del 1 al 9. Evalúa combinaciones específicas al presionar el botón "Ingresar":
  * Combinación `911` -> Muestra "password 1 correcto"
  * Combinación `714` -> Muestra "password 2 es correcto"
  * Cualquier otra combinación -> Muestra "password incorrecto"
* **Archivos:** `index3.html`, `script3.js`
